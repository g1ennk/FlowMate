### 1. 뽀모도로 타이머: 휴식 없이 완료 시 Flow 인정 안 되는 버그 수정 ✅

**문제**:
- 뽀모도로 타이머에서 Flow phase에서 완료했을 때, `MIN_FLOW_MS` 이상이어도 sessionHistory에 세션이 추가되지 않음
- 일반 타이머는 이미 수정되어 있었지만, 뽀모도로 타이머는 누락됨

**수정 내용**:
- **파일**: `frontend/src/features/timer/TimerFullScreen.tsx`
- **함수**: `handlePomodoroComplete`
- **변경 사항**:
  ```typescript
  // Flow phase에서 완료 시 sessionHistory 업데이트 로직 추가
  if (elapsedMs >= MIN_FLOW_MS && elapsedSec > 0) {
    newSessionHistory.push({ focusMs: elapsedMs, breakMs: 0 })
    // timerStore 및 sessionStorage에 반영
  }
  ```

**테스트 결과**:
- ✅ 뽀모도로 타이머에서 1분 1초 집중 후 완료 → sessionHistory에 세션 추가 확인
- ✅ 세션 상세에 "Flow 1: 1분 1초" 표시 확인

### 2. 추천 휴식 시간 계산: 최소/최대 제한 제거 ✅

**문제**:
- 추천 휴식 시간 계산에 최소 5분, 최대 30분 제한이 있어서 짧은 집중 시간에서도 5분이 추천됨
- 예: 3분 집중 → 36초 계산되지만 최소 5분으로 제한되어 5분 표시

**수정 내용**:
- **파일**: `frontend/src/features/timer/timerStore.ts`
- **함수**: `calculateBreakSuggestion`
- **변경 사항**:
  ```typescript
  // 이전: 최소 5분, 최대 30분 제한
  const clampedMin = Math.max(5, Math.min(30, suggestedMin))
  
  // 수정: 제한 없이 순수하게 20% 계산
  const suggestedMin = Math.round(focusMin * ratio)
  ```

**테스트 결과**:
- ✅ 3분 집중 → 1분 추천 (3분 × 20% = 0.6분 → 반올림 1분)
- ✅ 현재 세션만 기준으로 계산 (전체 누적 시간 X)
